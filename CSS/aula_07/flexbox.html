Alinhamento de Elementos e Conteúdo usando Flexbox

Flexbox é um mecanismo do CSS3 que permite aplicar facilmente o alinhamento e distribuição de elementos e conteúdos no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats e positions na resolução de dificuldades na construção de layouts.

As propriedades do Flexbox são as seguintes:

  • Flex-Direction: define qual direção o container deseja empilhar os itens (column, column-reverse, row e row-reverse);
  • Flex-Wrap: especifica se os itens flexíveis devem ser encapsulados ou não (flex-wrap: wrap, nowrap, wrap-reverse);
  • Flex-Flow: propriedade abreviada para definir Flex-Direction e Flex-Wrap (flex-flow: row wrap);
  • Justify-Content: alinhamento de itens (justify-content: center, flex-start, flex-end, space-around, space-between);
  • Align-items: Propriedade é usada para alinhar os itens flexíveis;
  • Align-content: Propriedade é usada para alinhar as linhas flexíveis.
